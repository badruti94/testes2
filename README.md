# Dokumentasi

## Provinsi
### url
``` http://127.0.0.1:8000/provinsi ```
### parameter
q | wajib | untuk memberikan keyword

### contoh
**request**
```
	http://127.0.0.1:8000/provinsi?q=jawa
```
**response**
```
	{
	    "keyword": "jawa",
	    "kategori": "provinsi",
	    "data": [
	        "Jawa Barat",
	        "Jawa Tengah",
	        "Jawa Timur"
	    ]
	}
```
